# 2Dタイピングシューティングゲーム「タイピングショット」 全体仕様書

## 1. ゲーム概要
ステージ構成型の2Dシューティングゲーム。タイピング要素を組み合わせたターゲット撃破システム。

## 2. 基本システム
### 2.1. 操作方法と表示
*   **ポインター移動**：マウスまたはトラックパッド操作。
    *   **照準表示（クロスヘア）**：ゲームステージ領域内では標準のカーソルを非表示とし、ポインターの現在位置に「十字のクロスヘア」を表示する。
    *   **プレイヤー機体 / 残弾システム**：どちらも不採用（プレイヤーキャラクターの機体表示や、残弾数・リロードの概念は一切なく、クロスヘアのみで狙うシンプルなエイム＆シューティングシステム）。
*   **撃破入力**：キーボード入力（アルファベット、数字、記号）およびマウスの各種クリック。

### 2.2. ステージルール
*   **ステージ数**：全5ステージ。
*   **制限時間**：各ステージ一律「60秒」。
*   **ゲーム領域**：横長の長方形。
*   **ヘッダーUI**：画面上部に現在の制限時間を表示。
*   **クリア条件**：制限時間内にステージ上の敵を全員倒す（時間差で出現する追加ターゲットを含む）。
*   **失敗条件**：
    *   制限時間以内に敵を全員倒せない。
    *   **ライフ喪失**：プレイ中にミス入力（ミスタイプ、ターゲット外クリック等）を**合計5回**行う（即時ゲームオーバー）。

### 2.3. ライフシステム
*   **初期ライフ**：5（❤️❤️❤️❤️❤️と表示）。
*   **ライフ減少条件**：ミスタイプ・誤操作をした場合（ターゲット外のクリック、誤ったキー入力、誤ったクリック操作）、ライフが1減少する。
*   **強制終了**：ライフが0になった（ミスが5回に達した）瞬間、即座にゲームは失敗終了となり、クリア失敗（リザルト画面）へ遷移する。

## 4. 追加システム・演出（コンボ＆ビジュアルフィードバック）
### 4.1. コンボシステム（Combo System）
*   **コンボ条件**：ミスタイプ・誤操作をせずに、連続してターゲット（的）を撃破すると「Combo」数がカウントアップされる。
*   **コンボリセット**：
    *   ターゲットのいない場所を誤ってクリックしたとき
    *   ターゲットに重なっているが、指示されたキーと異なるキーを押したとき
    *   ターゲットに重なっているが、指示されたクリックと異なるクリックを行ったとき
    *   これらのミス入力を行うと、コンボ数は即座に「0」にリセットされる。
*   **最大コンボ数（Max Combo）**：1ゲームの中で最も連続して撃破できたコンボ数を記録し、リザルト画面（クリア画面）に表示する。

### 4.2. ビジュアルフィードバック（Visual Feedback）
*   **撃破時エフェクト（Hit Particle）**：ターゲットの撃破に成功した瞬間、そのターゲットの座標を中心に、火花のような弾けるビジュアルエフェクト（パーティクル）を表示する。
*   **コンボポップアップ（Combo Popup）**：撃破成功時、撃破した座標から上方向へ「+1 Combo」や「5 Combo!」といったテキストが浮き上がりながらフェードアウトする演出を表示する。
*   **ミス・赤フラッシュ（Miss Flash & Popup）**：ミスタイプ・誤操作をした際、ゲームステージの外枠や領域が一瞬赤くフラッシュし、ミスした箇所に「MISS!」というテキストがポップアップする。

# 2Dタイピングシューティングゲーム「タイピングショット」 全体仕様書

## 1. ゲーム概要
ステージ構成型の2Dシューティングゲーム。タイピング要素を組み合わせたターゲット撃破システム。

## 2. 基本システム
### 2.1. 操作方法と表示
*   **ポインター移動**：マウスまたはトラックパッド操作。
    *   **照準表示（クロスヘア）**：ゲームステージ領域内では標準のカーソルを非表示とし、ポインターの現在位置に「十字のクロスヘア」を表示する。
    *   **プレイヤー機体 / 残弾システム**：どちらも不採用（プレイヤーキャラクターの機体表示や、残弾数・リロードの概念は一切なく、クロスヘアのみで狙うシンプルなエイム＆シューティングシステム）。
*   **撃破入力**：キーボード入力（アルファベット、数字、記号）およびマウスの各種クリック。

### 2.2. ステージルール
*   **ステージ数**：全5ステージ。
*   **制限時間**：各ステージ一律「60秒」。
*   **ゲーム領域**：横長の長方形。
*   **ヘッダーUI**：画面上部に現在の制限時間を表示。
*   **クリア条件**：制限時間内にステージ上の敵を全員倒す（時間差で出現する追加ターゲットを含む）。
*   **失敗条件**：制限時間以内に敵を全員倒せない。

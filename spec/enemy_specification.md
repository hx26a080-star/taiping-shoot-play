# エネミー（敵）仕様書

## 1. 基本ビジュアル
*   **外観**：赤い丸。
*   **サイズ**：出現するエネミーごとに、難易度（ステージ）に応じた「基準値」から上下します。
    *   **ステージごとの直径基準値**：
        *   **ステージ 1**：基準 120 ピクセル (エイムが容易)
        *   **ステージ 2**：基準 110 ピクセル
        *   **ステージ 3**：基準 100 ピクセル
        *   **ステージ 4**：基準 90 ピクセル
        *   **ステージ 5**：基準 80 ピクセル (エイムが困難)
    *   **基準値からのランダムな上下（ブレ）**：
        *   スポーンするエネミーごとに、各ステージの基準値から **±20% の範囲でランダムにサイズが上下** します。
        *   （例：ステージ 3（基準 100px）の場合、出現するエネミーの直径は 80px 〜 120px の範囲内でランダムに決定されます）
*   **テキスト表示**：円の中心部分に、ランダムに選ばれた1つのキーを表示。
    *   **フォント**：読みやすく、円からはみ出さないサイズ。

## 2. 出現（スポーン）と生存ルール
*   **出現位置**：ステージ（ゲーム領域）内のランダムな位置に直接出現（スポーン）する。
*   **出現時の演出**：特別なアニメーション等は行わず、直接出現する。
*   **画面端での挙動**：移動するエネミーがステージの境界（画面端・壁）に衝突した際、物理的に跳ね返り（バウンド）、ステージの外には出ない。
*   **生存（消滅）ルール**：エネミーが画面から消えるのは、プレイヤーが撃破した瞬間のみ。自爆や時間経過による自然消滅、画面外への離脱は行わない。

## 3. キー表示ロジック
*   **使用文字の範囲**：
    *   **キーボード入力**：アルファベット（A-Zまたはa-z）、数字（0-9）、記号（`-`, `[`, `]`, `;`, `:`, `,`, `.`, `/` など）
    *   **マウス操作（クリック）**：
        *   左クリック（表示：「左」）
        *   右クリック（表示：「右」）
        *   ホイールクリック（表示：「ホ」）
*   **重複許容**：複数のエネミーの間で、表示されるキー（またはマウスクリックアクション）が重複することを許容する（画面上に同じ入力指示を持つエネミーが同時に存在しても良い）。

## 4. 移動パターン
エネミーは以下の4つの基本移動パターンのいずれかを持つ。

| パターン名 | 挙動内容 |
| :--- | :--- |
| **静止型** | スポーン（出現）した位置から移動しない。 |
| **直線移動（横）** | 左右に往復、または画面端でバウンドするように横方向にのみ移動する。 |
| **直線移動（縦）** | 上下に往復、または画面端でバウンドするように縦方向にのみ移動する. |
| **自由移動（縦横）** | 斜めやカーブなど、縦・横両方のベクトルを組み合わせて画面内を自由に動く（画面端でバウンドする）。 |

## 5. 速度特性
移動するエネミーには、以下の2種類の速度特性がある。

1.  **等速型**：移動速度が一定（変化しない）。
2.  **変化型**：移動中に加減速したり、方向転換のタイミング等で一時的に速度が変化する。

## 6. 撃破（タイピング・クリック判定）仕様
*   **条件**：マウスカーソルまたはトラックパッドポインターが、対象のエネミー（赤い丸）の判定エリア（コライダー）に重なっている。
    *   **重なり時の優先順位**：複数のエネミーが同じ位置に重なっている場合は、最前面優先ではなく、マウスカーソルが重なっているすべてのエネミーのうち、入力が一致するものは重なり順に関わらずすべて同時に撃破されます。
*   **入力**：エネミーの上にポインターがある状態で、そのエネミーに表示されている指示に対応する入力を行う。
    *   **キーボードキーが表示されている場合**：対応する指定キーを押下。
    *   **「左」が表示されている場合**：マウスの左クリック。
    *   **「右」が表示されている場合**：マウスの右クリック（※ブラウザの標準右クリックメニュー起動はゲーム領域内で無効化する）。
    *   **「ホ」が表示されている場合**：マウスのホイールクリック。
*   **結果**：入力が一致するとエネミーが消滅し、撃破数にカウントされる。
